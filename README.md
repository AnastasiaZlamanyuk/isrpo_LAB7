## Описание работы
Данная лабораторная работа посвящена освоению адаптивной верстки с использованием технологии Flexbox. В ходе работы изучаются основные свойства и принципы построения гибких одномерных раскладок для создания адаптивных веб-интерфейсов. Практические задания включают создание навигационного меню, карточек товаров и центрирования элементов. В завершение выполняется самостоятельная верстка страницы с использованием изученных свойств Flexbox.
